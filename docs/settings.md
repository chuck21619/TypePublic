# Settings

The settings for the app is in the sidebar. activated by the default `**⌘,**`

## Folio

Show Icons

* Only show Folder Icons
* Turn off Special Folder Icons \(all folders are the same\)

## Workspace

* Outline
  * on/off
  * Show \#\#\#
* Line Numbers
  * On/off
  * Left or right side
* Theme
* Typography
  * Font
    * Mono
    * Sans
    * Serif
  * Indent the first line of a new paragraph
